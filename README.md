# Batch Projects

This repository contains a collection of automation tools and batch scripts. Each project is contained in its own directory with dedicated documentation.

## Available Projects

### 1. [Folder Cleanup Tool](FolderCleanup/README.md)

A modular PowerShell tool designed for safe and efficient directory cleaning.

- **Key Features**: Scope safety (depth limits), preset headers (empty files/folders), and keyword/extension filtering.
- **Location**: `/FolderCleanup`
